  # Installation
  User have to install:
  * [Node.js](https://nodejs.org/en/download/)
  * [Express](https://expressjs.com/)
  * [UUID](https://www.npmjs.com/package/uuid)

  # Technologies
  * JavaScript
  * Node.js
  * Express

  # Usage
  Clone this repo and run command
  ```
  npm i
  ```
  to install express and uuid
